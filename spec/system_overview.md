# Becoming System Overview

Becoming is a generative music engine designed to produce continuous evolving audio.

The system operates as a **sonic ecosystem**.

## Core Properties

- endless runtime
- layered audio fragments
- probabilistic evolution
- controlled repetition
- state-based sound behavior

## Major Components

1. Audio Library
2. Playback Engine
3. Scheduler
4. State Engine
5. Mutation Engine
6. Memory / Anti-Repetition System

## Runtime Loop

while running:

- evaluate current state
- update density
- select fragments
- schedule playback
- mutate structure
- update memory
