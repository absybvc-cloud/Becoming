# Time Model

Becoming operates across multiple time scales.

## Micro Time

30 seconds – 5 minutes

Events:
- fragment changes
- noise events
- tonal motifs

## Meso Time

10 – 30 minutes

Events:
- density shifts
- rhythm transitions
- tonal atmosphere changes

## Macro Time

1 – 3 hours

Events:
- state changes
- ecosystem transitions
- sonic climate changes
