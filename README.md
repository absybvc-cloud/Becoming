# Becoming — The Endless Album

Becoming is an endless generative album.

Unlike traditional albums, Becoming is not a collection of recordings.  
It is a system that continuously generates music.

Every listening session is unique.

The system never truly ends.  
Music continues to evolve as long as the system runs.

Listeners do not play tracks.  
Listeners enter a moment of the system.

## Core Concept

Music is no longer a fixed object.

Music becomes a living process.

Becoming is an experiment in:

- generative composition
- sound ecology
- endless musical systems
- time-based listening

## System Philosophy

Becoming behaves like a natural system:

- pulse and drift states
- slow macro evolution
- minute-scale micro variation
- open sonic ecosystem

The system evolves continuously.

## Project Structure
becoming/
src/

The `spec` directory contains design specifications used by the engine.

## MVP Goal

Build a system capable of generating coherent evolving music for **30–60 minutes** without obvious repetition.

Later versions will support:

- web runtime
- distributed worlds
- persistent album instances
